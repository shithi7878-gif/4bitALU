# 4bitALU
Simulation of Arithmetic Logic Unit using Proteus
## Project Overview
This project is a 4-bit Arithmetic Logic Unit (ALU) simulation designed and tested using Proteus. It can perform basic arithmetic operations (like addition, subtraction) and logical operations (like AND, OR, XOR).

### Hardware/Software Used
* **Simulation Tool:** Proteus Design Suite
* **Components:** 74LS series ICs (or logic gates according to your design)

### Project Files
* `new without 1seg.pdsprj` - Main Proteus simulation file.

### How to Run
1. Download the `.pdsprj` file from this repository.
2. Open it using Proteus Design Suite.
3. Click the **Run/Play** button at the bottom left corner to start the simulation.
